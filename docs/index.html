Named clipboard slots with global hotkeys

Save anything to numbered slots, recall it instantly with a keystroke. A lightweight CLI + daemon for macOS.

Install
GitHub
clipslots 
$ clipslots save 1
Saved to slot 1: "Hello, world!"

$ clipslots save 2
Saved to slot 2: "https://github.com/olafglad/clipSlots"

$ clipslots list
  1: Hello, world!
  2: https://github.com/olafglad/clipSlots

$ clipslots paste 1
Pasted from slot 1: "Hello, world!"

$ clipslots start
Daemon started. Global hotkeys active.
  Ctrl+Opt+[1-9]  → save to slot
  Ctrl+Cmd+[1-9]  → paste from slot

Everything you need, nothing you don't

ClipSlots is a single binary with zero dependencies. No Electron, no menubar app, no subscription.

Named Slots

Save clipboard contents to numbered slots (1-9). Each slot holds text, images, or rich content until you clear it.

Global Hotkeys

Save and paste with system-wide keyboard shortcuts. Works in any app, no focus switching needed.

Rich Clipboard

Preserves images, RTF, HTML, and file references — not just plain text. What you copy is what you paste.

Daemon Mode

Run as a background daemon with launchd integration. Starts on login, stays out of your way.

TOML Config

Customize hotkey bindings, slot count, and behavior with a simple config file. Hot-reloads on save.

CLI First

Full control from the command line. Script it, alias it, pipe it. Built for developers.

Terminal 
# Clone and build
git clone https://github.com/olafglad/clipSlots.git
cd clipSlots
swift build -c release
cp .build/release/clipslots /usr/local/bin/

# Set up and run
clipslots permissions
clipslots start
